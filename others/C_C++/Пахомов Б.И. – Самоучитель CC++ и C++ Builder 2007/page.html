Аннотация

Книга посвящена изучению программного продукта C++ Builder 2007 (в дальнейшем для краткости — Builder). Для читателя, только начинающего изучать эту среду программирования, нововведения, произведенные в Builder 2007 по сравнению с Builder 2006, останутся практически незаметными. Новичок даже может не отличить обе эти среды, так они похожи, внешне различия есть лишь в изображении центральной части рабочего стола (область Welcome Page).

Для начинающего программиста также не играет никакой существенной роли то, что в новом варианте: усовершенствован процесс компиляции и сборки проекта; при построении проекта можно создавать свои собственные конфигурации; C++-пакеты теперь можно компилировать в Delphi; добавлено несколько новых компонентов; из-за ограничений Microsoft перенесен Demo-файл из папки Program Files в новое место MyDocuments\RADStudio\Demos; изменилось месторасположение некоторых классов и названия их методов, а это, в свою очередь, привело к невозможности запуска приложений из ранних версий и к проблемам при ручной настройке таких приложений, т. к. функции, применявшиеся ранее, получили другую структуру и новые названия.
